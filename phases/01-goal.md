# Goal — RuslanOS

## Миссия
Создать репозиторий и инфраструктуру для проекта **RuslanOS** — минимальной ОС на базе LineageOS, где Ruslan Agent является единственным пользовательским интерфейсом и инструментом модификации системы.

## Ключевые цели (MVP)
1. Создать GitHub-репозиторий `valldun1/RuslanOS` с открытой лицензией (Apache 2.0)
2. Определить архитектуру: что форкаем, как встраиваем Руслана, как собираем
3. Написать план сборки: от GSI-прототипа до полного ROM
4. Подготовить документацию и CI для сборки
5. Первый артефакт: RuslanOS GSI (Generic System Image) с встроенным Ruslan Agent

## Критерии готовности
- [ ] Репозиторий создан и опубликован
- [ ] Архитектурный план утверждён
- [ ] README с описанием проекта
- [ ] Device tree определён (модель телефона)
- [ ] CI/CD для автоматической сборки
- [ ] Первый GSI-образ собран и протестирован

## Ограничения
- Сборка на Mac Mini (16GB RAM) — требуется оптимизация билда
- Устройство с Project Treble (для GSI)
- Лицензия Apache 2.0 (LineageOS) — сохранить attribution
