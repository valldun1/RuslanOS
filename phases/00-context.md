# Context — RuslanOS

## Что это
RuslanOS — минимальная операционная система на базе LineageOS (AOSP), где единственным системным интерфейсом является **Ruslan Agent**. Пользователь не возится с лаунчерами, настройками и пакетами — он говорит Руслану, и Руслан делает.

## Философия
- Минимум предустановленного: сеть (WiFi/LTE) + Руслан
- Пользователь модифицирует ОС через Руслана (голосом или текстом)
- Руслан = менеджер пакетов, панель настроек, терминал, техподдержка
- Никакого лишнего UI — только то, что нужно для работы с Агентом

## Платформа
- **Устройство:** второй телефон Valentin (модель уточняется — такая же как у него в руке)
- **Базовая ОС:** LineageOS 22.2 (Android 15) — Apache 2.0, можно форкать
- **Архитектура:** ARM64
- **Сборка:** планируется на Mac Mini M4 (16GB RAM) через OrbStack/UTM, либо на HES

## Текущий статус
- Valentin уже использует Ruslan Agent на планшете (через Telegram + Termux)
- Идея: второй телефон сделать выделенным RuslanPhone
- Варианты реализации: от простого (GSI + Magisk модуль) до полной сборки (свой ROM)
