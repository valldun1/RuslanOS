# RuslanOS — План разработки

Минимальная AI-операционная система на базе LineageOS, где **Ruslan Agent** — единственный интерфейс пользователя.

## 🎯 Философия

**Пользователь не возится с настройками — он говорит Руслану, и Руслан делает.**
Руслан = менеджер пакетов, панель настроек, терминал и техподдержка в одном лице.

## 🏗 Архитектура

```
RuslanOS
 ├── LineageOS 22.2 (Android 15) — минимальный AOSP
 ├── Ruslan Agent (/system/priv-app) — ядро системы
 ├── RuslanShell (Launcher) — минимальный UI/чат
 └── Magisk Module ruslanos-setup — установка + твики
```
