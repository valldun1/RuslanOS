# Архитектура RuslanOS

## Обзор

RuslanOS — минимальная AI-first операционная система на базе LineageOS (Android 15 AOSP).
Единственный пользовательский интерфейс — **Ruslan Agent** (через Telegram API или локальный WebView).

## Компоненты

### 1. Ruslan Agent (`/system/priv-app/RuslanAgent/`)

Ядро системы. Обеспечивает:
- Чат-интерфейс через Telegram
- Выполнение shell-команд на устройстве (root)
- Доступ к System API (Content Providers, Intents, Settings)
- LLM-логику (через opencode-go gateway)

**Не удаляется пользователем** — системное приложение.

### 2. RuslanShell (Launcher)

Минимальный лаунчер, заменяющий стандартный:
- Splash-экран (3 сек, логотип + «говори — и система сделает»)
- Интерфейс чата (поле ввода, пузырьки сообщений)
- Lock Task Mode (kiosk — нельзя выйти из лаунчера)
- Тёмная тема (адаптивный дизайн)

**Написан:** Kotlin + Jetpack Compose  
**Минимальный SDK:** 30 (Android 11)

### 3. Magisk-модуль (ruslanos-setup)

Набор скриптов для интеграции RuslanOS в существующую прошивку:

| Файл | Назначение |
|------|-----------|
| `customize.sh` | Пост-установочные твики |
| `service.sh` | Автозапуск агента при загрузке |
| `uninstall.sh` | Чистое удаление |
| `system/bin/ruslan` | CLI-команда вызова Руслана |
| `system/etc/init/ruslanos.rc` | Init-скрипт |

### 4. Скрипты сборки

| Скрипт | Назначение |
|--------|-----------|
| `scripts/setup-build-env.sh` | Развёртывание билд-среды |
| `scripts/build-full-rom.sh` | Полная сборка ROM |
| `scripts/build-gsi.sh` | Сборка GSI с патчами |
| `scripts/device-check.sh` | Проверка совместимости |
| `scripts/sign-ota.sh` | Подпись OTA-пакетов |

## Коммуникация между компонентами

```
[Пользователь] → Telegram API → Ruslan Agent
                                    ↓
                            CommandProcessor
                                    ↓
                            Shell (sh -c)
                                    ↓
                            System API / Filesystem
                                    ↓
                            Результат → Telegram → [Пользователь]
```

## Модификация через Руслана

Ключевая фича: пользователь не настраивает ОС руками — он говорит Руслану:

```
«Установи браузер»     → pm install ...
«Смени обои»           → settings put system wallpaper
«Какой статус?»        → dumpsys battery / df -h
«Заблокируй рекламу»   → install AdAway + настройка hosts
«Собери OTA»           → build-full-rom.sh --device <x>
```
