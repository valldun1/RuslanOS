# Команды Руслана в ОС

RuslanOS понимает естественный язык. Вот что можно сказать Руслану:

## Управление системой

### Установка приложений
```
«Установи Firefox»
«Установи Telegram»
«Install Aurora Store»
```
→ Руслан скачивает APK и устанавливает через `pm install`

### Запуск приложений
```
«Запусти Firefox»
«Открой настройки»
«Open Telegram»
```
→ `monkey -p <package> 1`

### Статус системы
```
«Статус»
«Что с батареей?»
«Сколько памяти?»
```
→ `dumpsys battery`, `df -h`, `uptime`

### Настройки
```
«Смени обои»           — (скоро)
«Сделай темнее»        — смена темы
«Включи WiFi»          — svc wifi enable
«Настрой яркость 50%»  — settings put system screen_brightness 50
```

## Работа с файлами
```
«Покажи файлы»                 — ls -la
«Создай папку projects»        — mkdir -p
«Найди файл config»            — find /sdcard -name '*config*'
«Скачай file.zip по ссылке»    — wget <url>
```

## Сеть
```
«Какой IP?»                — ip addr show
«Проверь связь с 8.8.8.8» — ping -c 4 8.8.8.8
«Скорость интернета»       — (скоро)
```

## Система
```
«Перезагрузи»
«Выключи»
«Reboot recovery»
```
→ `reboot`, `reboot recovery`

## Shell-команды
Любая строка, которую Руслан не распознал как команду, выполняется как shell:
```
«ls -la /system»
«ps aux | grep android»
«dumpsys cpuinfo»
```
