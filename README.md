ᴿᵁ RuslanOS
══════════════════

**Руслан — не просто помощник. Руслан — это операционная система.**

Минимальная AI-first операционная система на базе **LineageOS (Android 15)**.
Единственный интерфейс — **Ruslan Agent**. Ты говоришь — Руслан делает.

---

## 🔥 Философия

Современные мобильные ОС — это лаунчеры, виджеты, магазины приложений, настройки,
уведомления... РусланOS — это **другое**.

- **Минимум предустановленного:** сеть + Руслан
- **Никакого лишнего UI:** только то, что нужно для взаимодействия с Агентом
- **Модификация через Агента:** хочешь браузер? — «Руслан, поставь браузер».
  Хочешь сменить обои? — «Руслан, сделай красиво».
  Хочешь новую прошивку? — «Руслан, собери OTA с моими изменениями».

**Руслан = менеджер пакетов + панель настроек + терминал + техподдержка.**

## 🚀 Как попробовать

```bash
# 1. Разблокировать bootloader на устройстве
# 2. Прошить LineageOS GSI:
fastboot flash system lineage-22.2-xxxx.img
fastboot -w
# 3. Установить Magisk
# 4. Установить модуль ruslanos-setup
# 5. Перезагрузить — и ты в RuslanOS
```

*Подробная инструкция появится в `/docs`*
